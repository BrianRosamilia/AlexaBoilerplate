# Find my iPhone Alex Skill

## Setup

Create a file called `config/default.json` and fill in the required fields:

```js
{
	"Amazon" : {
		"lambda-arn" : "required",
		"profile" : "optional"
	},
	"Apple" : {
		"username": "required",
		"password": "required",
		"device": "required-name of your iOS device"
	}
}

```
